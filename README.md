# bash_rpg

Features:

4 character classes (Warrior, Mage, Rogue, Paladin)
Full leveling system with stat progression
Turn-based combat with skills and items
15+ enemy types across 4 exploration zones
Boss battles
Complete inventory and equipment system (weapons, armor, accessories)
Shop with buyable/sellable items
Quest system with rewards
Inn for healing
Save/load functionality
Colored terminal interface
